## 3. 教练技术，不一定"标准"地完整应用
**ID**: 1905089011433746200

### 优化后
```
**常见误区**

误区1：和违章员工沟通时，难以从教练技术理论自然转换到实际家庭身份场景，容易放不开

误区2：认为完整的教练技术框架必须从头用到尾，每一步都不能省略


**核心解答要点**

**不需要强求完整步骤**

GROW教练模型只需用到其中一个点，就能对沟通起到帮助

就像打球不需要每次都打出所有技术动作，关键时刻能用对一招就有价值


**灵活调整沟通开场方式**

不一定要用提问方式，也可以直接确立共同目标

例如开场直接说「我不希望你受伤，你肯定也不希望」

可以快速和员工建立共同目标，降低对方的防备心


**沟通提问的实用技巧**

对方提出问题时，可以问：如果成功了，应该是个什么样子？

讨论解决方案时，可以问：如果不考虑限制条件，咱们还有什么方法可以用？


**基础梳理同样有效**

很多时候心平气和把现状梳理清楚，就对沟通很有帮助


**关键总结**

GROW用到一个点，对原来就有帮助

就好比打球，你学了抢攻正手反手上旋下旋

比赛的时候关键时刻用上一招能多得一分那就用

不能完整展示你的技能也没关系
```

## 4. [模型] 信任公式：可信可靠亲近
**ID**: 1904960661704679200

### 优化后
```
**信任公式**

**信任 = 可信度 + 可靠度 + 亲近感 - 自我导向**


**可信度**：我相信他说的话

反面：信口开河


**可靠度**：我相信他会行动

反面：不靠谱


**亲近感**：我相信他关心我的利益

反面：公事公办


**自我导向**：他只关心自己的利益

反面：以客户为中心
```
