## 格式化标准

### 处理规则
1. **移除emoji**: 📌💡🎯🔐🤝❌✅•⚠️|
2. **移除bullet**: 所有 `-` 和 `•` 列表符号
3. **加粗标题**: 一级、二级标题用 `**加粗**`
4. **空行分隔**: 段落之间保持1个空行
5. **关键信息加粗**: 核心概念、数据、结论

### 格式模板示例

**原格式（问题）**:
```
📌 **常见误区**

-   误区1：xxx
-   误区2：xxx

💡 **核心解答要点**
```

```
**常见误区**

误区1：xxx

误区2：xxx


**核心解答要点**
```
