# Get笔记标题生成提示词

## 使用方法
把下面这段提示词发给 Kimi（Kimi Claw、网页版、或我瓜瓜都行），然后把笔记内容贴给它。

---

## 提示词正文（复制这段）

```
你是Get笔记标题生成专家。

任务：为笔记生成标题，要求一眼能看明白"是什么事、什么观点"。

输出格式：
- 15-25字
- 结构：【主题】+【核心洞察/结论】
- 不用完整句子，用短语

示例：
❌ 不好：今天14号还没开学唐浩瑞和唐浩瑄一整天没做多少事
✅ 好：孩子假期学习状态：除法学不会却自认为掌握

❌ 不好：我们可以从这几个核心维度判断职业健康体检机构的服务质量
✅ 好：职业体检质量判断：主检医师资质是核心

现在请为以下笔记生成标题：

[PASTE_YOUR_NOTE_HERE]
```

---

## 快捷方式

把上面提示词保存为快捷短语（如果你用iPhone）：

**设置 → 通用 → 键盘 → 文本替换**
- 短语: （上面整段提示词）
- 输入码: `gettitle`

以后输入 `gettitle` 就会自动展开。

---

## 批量处理版

如果一次要处理多条笔记：

```
请为以下每条笔记生成标题（格式：序号. 标题）：

要求：
1. 一眼能看出"是什么事"
2. 一眼能看出"核心观点/结论"
3. 15-25字，简洁有力

笔记列表：
---
1. [笔记1内容]
2. [笔记2内容]
3. [笔记3内容]
```
